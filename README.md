# Telegram PPTX Bot

Бот для генерації PowerPoint-презентацій зі сценаріїв (Telegram + python-pptx).

## Запуск локально

```bash
pip install -r requirements.txt
cp .env.example .env   # створити .env
# вписати свій BOT_TOKEN в .env
python bot.py
